# Personal Portfolio Website

## Project Overview

This is my personal portfolio website built with HTML and CSS to showcase my professional profile, skills, and contact information. It serves as a central place to present my background, technical abilities, interests, and ways to connect with me.

## Features

- **Introduction Section:** Brief personal introduction with a professional tone.
- **Skills Showcase:** Displays technical skills in programming and growing competencies.
- **Contact Section:** Includes links to LinkedIn and other ways to get in touch.
- **Responsive Layout:** Simple and clean design focused on usability and clarity.

## Technologies Used

- HTML  
- CSS  
- Images and icons for visual enhancement  

## Purpose

This site is intended as a **digital CV/portfolio** to present my experience and skills to recruiters, hiring managers, and peers. It complements my GitHub profile and LinkedIn presence by offering a web‑friendly summary of my professional background.

## How to View

1. Open the file `index.html` in a browser.  
2. You can also host this project on GitHub Pages for public access:
   - Go to your repository settings
   - Enable GitHub Pages from the `main` branch
   - Access the published site at `https://pablohv91.github.io/cv1`

## What You Will Find

- Personal introduction and current focus  
- Description of technical skills in Python, JavaScript, HTML and CSS  
- Growing knowledge in databases (SQL, NoSQL) and frameworks such as Flask and NodeJS  
- Personal interests and professional aspirations
